# PORTFOLIO

# BS Portfolio – Web Developer

This repository contains my personal portfolio website, developed using modern frontend technologies to demonstrate my skills, projects, and practical experience in web development. The application is designed with a strong focus on performance, clean architecture, and high-quality user interface design.

---

## Overview

I am BS, a Full Stack Web Developer with a focus on building scalable, responsive, and visually refined web applications. This portfolio represents my approach to development, combining technical proficiency with attention to design and user experience.

---

## Key Features

* Single-page application built with React (Vite)
* Clean and modular component-based architecture
* Advanced UI effects including aurora background, chroma grid, and blur text animations
* Smooth transitions and interactions powered by Framer Motion
* Fully responsive design across devices and screen sizes
* Optimized performance and fast load times

---

## Technology Stack

* Frontend: React, Tailwind CSS
* Animation: Framer Motion
* Build Tool: Vite
* Languages: JavaScript, HTML, CSS

---

## Application Structure

The portfolio is structured as a single-page interface with the following sections:

* Hero Section: Central profile image with introduction
* About: Summary of background and development focus
* Skills: Technical competencies and tools
* Projects: Selected work with descriptions
* Contact: Communication and professional links

---

## Deployment

The application can be deployed using platforms such as Vercel, Netlify, or any modern hosting service that supports Vite-based builds.
